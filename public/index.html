How points work

Point Tiers
🫧 Laundry Amendment

Points are awarded per stage. Whoever completes each stage logs it independently. Starting a wash and walking away does not entitle you to fold-and-put-away credit.

StagePtsNotes
🫧 Wash + move to dryer1ptStart machine and transfer when done
👕 Fold1ptActually folding — not leaving in the dryer
🗂️ Put Away1ptInto drawers/closet — not the bedroom chair

Full cycle (all 3 stages, one person) = 3pts total.

🏋️ Heavy Lifting Rule

If both people work together on a 3pt task, both log 3pts independently. Shared effort is not penalized.

Rules of Engagement
  • Same-day logging only — log at time of completion. No retroactive entries from "last Tuesday."
  • No self-awarding bonus points — points are fixed per category. You cannot upgrade a 1pt task to 3pt because you really didn't want to do it.
  • Minimum effort standard — a chore only counts if completed. Loading the dishwasher and leaving it unstarted does not count as "did the dishes."
  • Laundry chair rule — put-away only counts if laundry reaches drawers/closet, not the bedroom chair.
  • School drop-off credit — only awarded if the child arrives on time with all required items. No partial credit.
  • Week resets Monday midnight — no points carry over. Each week is a fresh start.
  • No sabotage — deliberately making messes to score more cleanup points is grounds for immediate disqualification and a serious conversation.
Dispute Resolution
  • The app is the ref — if it isn't logged, it didn't happen.
  • Contest within 24 hours — disputes must be raised within 24 hours of the entry being logged.
  • School drop-off disputes — if the child is late or missing items, the logging party forfeits those points. No exceptions.
  • Ties = takeout — in the event of a tie, both parties win. Celebrated with takeout, split evenly.
